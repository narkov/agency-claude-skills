# Website Launch Checklist

Run through a comprehensive pre-launch checklist for a website.

## Checklist

### Technical
- [ ] SSL certificate installed and working
- [ ] All pages return 200 (no broken links/404s)
- [ ] Redirects from old URLs configured (if redesign)
- [ ] robots.txt allows crawling
- [ ] XML sitemap generated and submitted
- [ ] 301 redirects from www/non-www
- [ ] Custom 404 page
- [ ] Favicon and app icons

### SEO
- [ ] Unique title tags on all pages (50-60 chars)
- [ ] Meta descriptions on all pages (150-160 chars)
- [ ] H1 tags on every page (single H1)
- [ ] Image alt texts
- [ ] Schema markup (Organization, LocalBusiness, BreadcrumbList)
- [ ] Open Graph and Twitter Card tags
- [ ] Canonical URLs set
- [ ] Hreflang tags (if multilingual)

### Performance
- [ ] Images optimized (WebP, lazy loading)
- [ ] CSS/JS minified
- [ ] Gzip/Brotli compression enabled
- [ ] Core Web Vitals passing (LCP, FID, CLS)
- [ ] CDN configured

### Analytics & Tracking
- [ ] Google Analytics 4 installed
- [ ] Google Tag Manager configured
- [ ] Conversion events set up and firing
- [ ] Google Search Console verified
- [ ] Facebook Pixel (if running ads)

### Legal
- [ ] Privacy policy page
- [ ] Cookie consent banner
- [ ] Terms of service (if applicable)

## Usage

Provide the site URL. I'll check what I can automatically (via browsing/crawling) and create a status report with passed/failed/manual-check items.
