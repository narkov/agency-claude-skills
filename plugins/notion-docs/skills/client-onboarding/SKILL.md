# Client Onboarding

Set up a complete Notion workspace for a new client project.

## Required Info

- Client name
- Website URL
- Services: SEO / Ads / Web Development (which ones)
- Primary contact
- Start date

## Steps

1. Create a new project in Notion Projects database with:
   - Name: [Client Name]
   - Status: active
   - Type: client
   - Area: work
   - Priority: based on contract value
   - Notes: services scope, website URL, contact info

2. Create sub-pages under the project:
   - **Brief** — client goals, target audience, competitors, KPIs
   - **Access & Credentials** — GA4, GSC, CMS, hosting (remind to use secure sharing)
   - **Strategy** — SEO plan / Ad plan / Dev plan depending on services
   - **Reports** — monthly report archive
   - **Meeting Notes** — ongoing notes

3. Set up initial tasks in the Tasks database:
   - Access collection (GA4, GSC, CMS credentials)
   - Initial audit (SEO/site/ads depending on scope)
   - Strategy document draft
   - First monthly report date

## Output

Confirm what was created with links to each Notion page.
