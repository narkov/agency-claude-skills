# Project Status Update

Update a client's project in Notion with current status, completed work, and next steps.

## Steps

1. Find the project in Notion by client name
2. Update project properties:
   - Status (if changed)
   - Next Step (brief description of what's next)
3. Add a note to the project page with:
   - Date
   - What was done since last update
   - Current blockers (if any)
   - Next actions with deadlines

## Input

Provide: client name, what was done, what's next. I'll format and update Notion.

## Output

Confirm the update was saved with a summary of changes.
